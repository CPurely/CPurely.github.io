plotnine的再次踩坑

2019-01-02
极客
约 192 字 预计阅读 1 分钟 次阅读

文章目录

论文画图需要,今天又一次安装plotnine画图试试,忘记了之前怎么使用的,结果默认的示例代码居然没法显示图像!!! 示例代码如下:

import pandas as pd
import numpy as np
from plotnine import *
from plotnine.data import *


(ggplot(mtcars, aes('wt', 'mpg'))
 + geom_point())

但是使用ipython可以显示.回想了以前使用plotnine的时候好像print了一下,于是进行了以下尝试:

1

import pandas as pd
import numpy as np
from plotnine import *
from plotnine.data import *


p = (ggplot(mtcars, aes('wt', 'mpg'))
 + geom_point())
print(p)

终于可以显示图像了.
