在autodesk官网注册帐号并下载linux版maya

通过注册Autodesk的教育版帐号，通过填写相关信息，会有一封包含产品密钥和序列号的邮件发到你的注册邮箱之中，我的使用年限是3年，足够使用。 下载好了压缩包，解压然后运行里面的setup即可安装 但是安装好之后有可能不能运行，因为缺少相关的libraries

1
2
3
4

/usr/autodesk/maya2017/bin/maya.bin: error while loading shared libraries: libXp.so.6: cannot open shared object file: No such file or directory
/usr/autodesk/maya2017/bin/maya.bin: error while loading shared libraries: libtiff.so.3: cannot open shared object file: No such file or directory
/usr/autodesk/maya2017/bin/maya.bin: error while loading shared libraries: libssl.so.10: cannot open shared object file: No such file or directory
/usr/autodesk/maya2017/bin/maya.bin: error while loading shared libraries: libcrypto.so.10: cannot open shared object file: No such file or directory

我的系统运行maya缺少这么几个libraries 但是我在系统中搜索这几个libraries其实都有，因此我直接将系统已有的libXp.so.6 libtiff.so.3 libssl.so.10 libcrypto.so.10 复制到/usr/autodesk/maya2017/lib即可 现在就可以打开maya了
