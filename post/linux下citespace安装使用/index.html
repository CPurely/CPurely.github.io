配置Linux安装citespace的环境

安装java环境与.7z解压工具

下载并安装java因为我用的是openSUSE因此下载rpm包安装即可，安装好了在terminal中运行以下命令检查一下java是否可以被正常调用

java -version

安装.7z解压工具

zypper in p7zip

下载citespace压缩文件

首先前往citespace下载最新的.7压缩文件

解压与运行

使用cd命令转移到压缩文件目录执行下面的命令解压文件

7z x filename.7z

cd进入解压后的文件夹，使用以下命令执行CiteSpaceV.jar

java -jar CiteSpaceV.jar

到此即可运行citespace
