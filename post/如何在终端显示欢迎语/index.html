如何在终端显示欢迎语

2020-03-29
极客
约 337 字 预计阅读 1 分钟 次阅读

文章目录

以前就看到有的人的终端登录的时候会有一段欢迎的字符画，很好奇是什么。然后这两天看到有人介绍怎么弄，我一时心血来潮也弄了一下，终于我的终端也可以显示字符画欢迎语啦！

为了达到这个效果，首先我们需要安装一个终端软件figlet，因为我用的mac所以使用以下命令安装：

1
2
3
4
5

# macos
brew install figlet

# ubuntu应该是
apt-get install figlet

安装完之后你可以简单试试这个命令行软件。

1

figlet hello world

输入以上命令你就能在终端上得到一幅hello world的字符画，很有意思，同时你也可以设置输出字体的样式。

为了能在启动的时候输出欢迎语，我们可以在zsh的配置文件里面加上：

1

figlet Welcome

然后使用source命令使配置生效。

1

source .zshrc

这样我们每次登录的时候就会出现welcome的欢迎语啦！
